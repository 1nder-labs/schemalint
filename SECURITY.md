# Security Policy

## Supported Versions

Only the latest release receives security fixes.

| Version | Supported |
| ------- | --------- |
| latest  | Yes       |
| older   | No        |

## Reporting a Vulnerability

**Do not open a public GitHub issue for security vulnerabilities.**

Please report vulnerabilities privately via GitHub Security Advisories:
https://github.com/1nder-labs/schemalint/security/advisories/new

Include a description of the issue, steps to reproduce, and any potential impact. You will receive a response on a best-effort basis, typically within a few business days.

## Disclosure

Confirmed vulnerabilities are fixed and shipped in a patch release. The advisory is made public after the fix is available.
